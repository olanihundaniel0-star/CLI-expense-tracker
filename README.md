# CLI Expense Tracker 

A Python-based command-line tool to track daily expenses. This project demonstrates file handling (CSV), data persistence, and input validation in Python.

## Features
* **Add Expenses:** fast entry with automatic date stamping.
* **View Summary:** neatly formatted table view of all spending.
* **Total Calculation:** automatically sums up expenses.
* **Data Persistence:** saves data to a local CSV file so records aren't lost on exit.

## How to Run
1.  Ensure you have Python installed.
2.  Clone this repository.
3.  Run the script:
    ```bash
    python tracker.py
    ```
